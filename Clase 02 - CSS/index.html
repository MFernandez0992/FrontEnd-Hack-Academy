Tipo de selectores en CSS

  • Element

    • Esto selecciona todos los elementos p en el documento.

  • Class

    • Un elemento puede tener un único id pero múltiples class, lo cual permite tener gran flexibilidad si se usa correctamente.

  • ID

    • Con el numeral (#) se le indica a CSS que un selector es de tipo id.Esto selecciona el (único) elemento HTML que tenga id=pepito.No importa qué elemento (etiqueta) HTML sea. En este ejemplo se trata de un p

  • Pseudo classes

    • Esto selecciona sólo los elementos del tipo seleccionado con class especificas del usuario.
